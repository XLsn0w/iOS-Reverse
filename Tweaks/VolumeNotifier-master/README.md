VolumeNotifier
==============

Play sound effect when changing volume.

Building:

Make sure $THEOS or $iOSOpenDev is set up correctly, and you have Xcode as well as the iOS 7+ SDK.

Build for profiling > Run BuildPackage.sh with root.
