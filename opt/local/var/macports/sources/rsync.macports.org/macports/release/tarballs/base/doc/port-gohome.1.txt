port-gohome(1)
==============

NAME
----
port-gohome - Open the homepage of the given port(s)

SYNOPSIS
--------
[cmdsynopsis]
*port* [*-D* 'portdir'] *gohome*
     [['portname' | 'pseudo-portname' | 'port-expressions' | 'port-url']]

DESCRIPTION
-----------
*port gohome* opens the homepage of the given port(s) in your default browser.

--------
man:port[1]
