portstyle(7)
============

NAME
----
portstyle - style guide for Portfiles

DESCRIPTION
-----------
This man page previously documented the style of Portfiles. The canonical source
for the formatting of Portfiles is now the guide:[The MacPorts Guide], which
also contains a section on guide:development.practices.portstyle[Port Style].
