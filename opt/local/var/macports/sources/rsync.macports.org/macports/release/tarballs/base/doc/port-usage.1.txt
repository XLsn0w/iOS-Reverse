port-usage(1)
=============

NAME
----
port-usage - Show a short command synopsis summary for the port command

SYNOPSIS
--------
[cmdsynopsis]
*port* *usage*

DESCRIPTION
-----------
*port usage* displays a very brief summary of all possible options of
man:port[1] and the canonical order in which port expects them. It prints the
*SYNOPSIS* port of the man:port[1] manpage.

SEE ALSO
--------
